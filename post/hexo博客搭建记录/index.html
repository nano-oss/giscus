Hexo博客搭建记录

  • Sun, Feb 18, 2024
  • One-minute read

参考链接

https://blog.csdn.net/weixin_45019350/article/details/121901433 https://blog.csdn.net/qq_43580193/article/details/117341489 https://zhuanlan.zhihu.com/p/137338730#

操作步骤：

1.安装环境依赖

下载最新的nodejs手动安装：https://nodejs.org/en/download/ wget https://nodejs.org/dist/v20.11.0/node-v20.11.0-linux-x64.tar.xz sudo mkdir -p /usr/local/lib/nodejs sudo tar -xJvf node-v20.11.0-linux-x64.tar.xz -C /usr/local/lib/nodejs

添加环境变量 用户.bashrc 系统/etc/profile

sudo nano /home/zhb/.bashrc sudo nano /etc/profile

都要在最后加入下面这行

export PATH=/usr/local/lib/nodejs/node-v20.11.0-linux-x64/bin:$PATH source .bashrc source /etc/profile

node -v # 有显示版本信息即为成功

安装git： sudo apt-get install git-core

安装cnpm(非必须): npm install -g cnpm –registry=https://registry.npm.taobao.org

2.安装hexo-cli npm install -g hexo-cli 或者 cnpm install -g hexo-cli

验证： hexo -v

3.初始化hexo hexo init myblog

这里大概率会出现git clone失败的错误

改一下这个文件init.js,把原来的git url替换一下：

const GIT_REPO_URL = ‘https://gitee.com/naonano/hexo-starter.git'

sudo nano /usr/local/lib/nodejs/node-v20.11.0-linux-x64/lib/node_modules/hexo-cli/dist/console/init.js 之后重新执行上面 hexo init

4.安装npm cd myblog npm install

5.启动站点 hexo g hexo server

6.在gitee搭建站点 gitee上创建仓库，仓库名称随意，路径设置为：用户名+.gitee.io 例如： binbyzhang.gitee.io

安装一个插件hexo-deployer npm install hexo-deployer-git –save

修改hexo配置文件指定仓库路径 sudo nano _config.yml 添加以下内容： deploy: type: git repo: https://gitee.com/naonano/binbyzhang.gitee.io.git branch: master

推送站点文件到gitee git config –global user.email “2231637869@qq.com” git config –global user.name “naonano” git config –global user.password " " hexo d 输入gitee账号和密码 然后开启仓库的 gitee pages服务

7.访问测试 https://naonano.gitee.io/binbyzhang.gitee.io/
