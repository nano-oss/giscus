香橙派驱动树莓派5-7寸触摸屏 系统最好是ubuntu

klipperscreen官方网址 https://klipperscreen.readthedocs.io/en/latest/

正常安装klipper、fluid、moonraker

git clone https://gitee.com/miroky/kiauh.git

安装桌面驱动： sudo apt install xserver-xorg-input-libinput

安装中文字库： sudo apt install fonts-wqy-zenhei

安装摄像头驱动，需要先卸载meg数据流重装、安装klipperscreen bash <(curl -fsSL http://klipper.7130404.xyz:8001/add_webcam.sh)

重启
