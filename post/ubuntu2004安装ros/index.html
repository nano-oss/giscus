Ubuntu2004安装ros

  • Wed, Sep 20, 2023
  • 3-minute read

ubuntu不同版本对应的ros版本：

Ubuntu ROS1.0 ROS 2.0 16.04LTS Kinetic LTS Ardent 18.04LTS Melodic LTS Dashing LTS 20.04LTS Noetic LTS Foxy LTS 22.04LTS Noetic Ninjemys Humble Hawksbill

1.创建一个ros用户 sudo adduser ros （sudo userdel -r username） sudo usermod -a -G adm ros sudo usermod -a -G sudo ros sudo gpasswd –add ros dialout su - ros

2.修改镜像源 sudo mv /etc/apt/sources.list /etc/apt/sources.list.bak sudo nano /etc/apt/sources.list 写入以下内容覆盖原文件，根据系统选择： Ubuntu 16.04 (LTS)代号为xenial。 Ubuntu 18.04 (LTS)代号为bionic。 Ubuntu 20.04 (LTS)代号为focal。

3.增加ros密钥 sudo nano /etc/apt/sources.list.d/ros-latest.list ,加入以下内容： deb https://mirrors.tuna.tsinghua.edu.cn/ros/ubuntu/ focal main

sudo apt-key adv –keyserver ‘hkp://keyserver.ubuntu.com:80’ –recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

sudo apt-get update

4.安装ros

sudo apt-get install ros-noetic-desktop-full

sudo apt-get install aptitude sudo aptitude install ros-noetic-desktop-full sudo apt install python3-pip sudo pip3 install rosdepc

5.初始化

sudo apt install python3-roslaunch

sudo apt install ros-noetic-roslaunch sudo rosdepc init sudo rosdepc update

6.配置环境变量 echo “source /opt/ros/noetic/setup.bash” » ~/.bashrc source ~/.bashrc sudo apt-get install python-rosinstall python-rosinstall-generator python-wstool build-essential

7.测试验证 roscore

新建终端： rosrun turtlesim turtlesim_node
